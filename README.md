# KNN and Decision Tree Classification

## Overview

This project demonstrates the implementation and comparison of two popular machine learning classification algorithms: K-Nearest Neighbors (KNN) and Decision Tree Classifier.

The objective is to build classification models, evaluate their performance, and compare their effectiveness on the given dataset using Python and Scikit-Learn.

---

## Objectives

The main objectives of this project are:

* Understand K-Nearest Neighbors (KNN)
* Understand Decision Tree Classification
* Perform data preprocessing
* Train classification models
* Compare model performance
* Evaluate prediction accuracy
